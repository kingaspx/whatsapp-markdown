# WhatsApp Markdown
> Javascript Function to Replicate WhatsApp Text Formatting Behaviour.

[![NPM Version][npm-image]][npm-url]
[![Downloads Stats][npm-downloads]][npm-url]

## Installation

```sh
npm install whatsapp-markdown --save
or
yarn add whatsapp-markdown
```
